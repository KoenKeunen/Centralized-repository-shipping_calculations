# Simple Interest Calculator Project

Dit project demonstreert het gebruik van Git en GitHub en bevat de code voor een eenvoudige rekenmachine voor enkelvoudige rente.

## Formule
De formule voor enkelvoudige rente (Simple Interest, SI) is:
$I = P \times R \times T$

Waarbij:
* I = Totale Rente
* P = Hoofdsom (Principal)
* R = Jaarlijkse Rente Percentage (Rate)
* T = Tijd in Jaren (Time)

## Bestandsstructuur
* `index.html`: Bevat de interface.
* `calculator.js`: Implementeert de renteberekening.
